## 1. Falsifiable Claim

A retrieval-augmented generation (RAG) system with **episodic memory consolidation**—where intermediate research steps are compressed via gradient-based summarization and stored in a structured knowledge graph—outperforms append-only log baselines on multi-step scientific reasoning tasks by ≥15% on answer accuracy, given fixed token budgets across 100+ step trajectories.

---

## 2. Experiment / Proof Harness

**Task:** Reproduce a subset of 50 problems from the *ScienceWorld* benchmark (or SWE-bench Lite) requiring ≥20 tool-using steps.

**Setup:**
- **Treatment:** Agent writes to a compressing memory module (fine-tuned T5-small for episodic summarization, + Neo4j graph for relational retrieval with BM25 + dense hybrid search).
- **Control:** Append-only text log with identical retrieval architecture.
- **Metric:** End-task accuracy / patches generated correctly; secondary: retrieval precision@5 for relevant past steps.

**Validation:** Open-source harness at `github.com/[org]/research-memory-harness` with Docker, fixed random seeds, and GPT-4-turbo as base planner to isolate memory effects.

---

## 3. Failure Modes

| # | Failure Mode | Detection |

| **B** | **Graph retrieval returns structurally similar but semantically wrong precedents** (e.g., same reagent, different temperature). | Adversarial test set with near-miss distractors. |
| **C** | **Memory maintenance overhead exceeds token savings**—consolidation latency causes planner timeout or truncated context. | Profile wall-clock time vs. effective context window utilization. |
