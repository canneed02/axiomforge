## 1. Falsifiable Claim

A retrieval-augmented generation (RAG) system with **episodic memory consolidation**—where intermediate research steps are compressed via gradient-based summarization and stored in a structured knowledge graph with temporal decay—outperforms naive append-only context baselines on long-horizon scientific task sequences (>50 steps) by ≥15% on answer accuracy, while using ≤30% of the token budget.

---

## 2. Experiment / Proof Harness

**HARNESS-MEM-1**

| Component | Specification |
|-----------|-------------|
| **Benchmark** | Custom suite: 200 synthetic multi-step scientific reasoning chains (e.g., "design PCR primers → check off-targets → order → troubleshoot gel") derived from Bio-protocol and Methods in Enzymology |
| **Baselines** | (a) Naive full-context append; (b) Fixed-size sliding window; (c) Oracle human summary |
| **Treatment** | Episodic RAG with: (i) gradient saliency for step compression, (ii) knowledge graph (Neo4j) with TTL decay, (iii) retrieval via dense + sparse hybrid |
| **Metrics** | Task success rate; hallucination rate (LLM-judge + human audit); cumulative tokens; retrieval precision@k |
| **Infrastructure** | Reproducible via Docker; all prompts, graph schemas, and evaluation scripts versioned |

## 3. Failure Modes

| ID | Failure Mode | Detection |
|----|-------------|-----------|
