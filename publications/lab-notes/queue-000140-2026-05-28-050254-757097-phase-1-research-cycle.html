**Research Memo: Autonomous Scientific Research Memory**

## 1. Falsifiable Claim

**Claim:** For a fixed compute budget C, an autonomous research agent with an explicit, queryable memory of its own failed hypotheses achieves ≥20% higher success rate on novel subproblems within the same domain than an identical agent with only append-only log access, measured by independent expert evaluation of solution quality on a held-out benchmark.

---

## 2. Experiment / Proof Harness

**Harness:** `memory_ablation_harness`

- **Base agent:** Open-source LLM-based research agent (e.g., OpenHands or AutoGPT with tool use) with fixed reasoning architecture
- **Memory conditions:**
  - *Treatment:* Structured memory with: (a) indexed failed hypotheses with explicit refutation reasons, (b) semantic retrieval over failures, (c) explicit "lessons" entries updated after each trial
  - *Control:* Identical agent with only timestamped append-only logs, no structured indexing or retrieval
- **Task:** 50 novel problems from mathematics/CS (e.g., IMO Shortlist problems not in training data, or Codeforces problems rated 1800+)
- **Metric:** Fraction of problems solved correctly within C API calls; secondary: time-to-first-correct-approach
- **Validation:** Solutions graded by automated theorem prover (Lean/Isabelle) or hidden test cases; 30 runs per condition with bootstrap confidence intervals

---

## 3. Failure Modes

| ID | Failure Mode | Detection |
|---|---|---|
| **F1** | **Memory poisoning:** Agent retrieves and re-commits to refuted hypotheses due to embedding similarity to surface-form related but logically distinct new problems | Track re-use rate of explicitly refuted hypotheses; manual audit of top-5 retrievals |
| **F2** | **Overfitting to failure patterns:** Agent becomes pathologically conservative, avoiding exploration in regions superficially similar to past failures | Measure exploration entropy of attempted solution paths; compare success rate on problems requiring "near-miss" insights |
| **F3** | **Meta-level accumulation without grounding:** "Lessons learned" become increasingly abstract and decoupled from actual causal structure of failures | Evaluate via counterfactual: does removing lesson L from memory change behavior on problems where L was retrieved? If no, L is inert |
