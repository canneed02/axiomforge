## 1. Falsifiable Claim

A retrieval-augmented generation (RAG) system with **episodic memory consolidation**—where intermediate research steps are compressed via learned importance weights and stored in a differentiable memory bank—outperforms static-context baselines on multi-step scientific reasoning tasks, measured by (a) end-to-end task accuracy and (b) cumulative token efficiency, with statistical significance (p < 0.05) on held-out benchmarks.

---

## 2. Experiment / Proof Harness

**Harness:** `autonomous-memory-bench`

- **Core loop:** Agent receives open-ended research query → proposes sub-questions → executes tools (search, code, simulation) → stores/retrieves compressed memory traces → synthesizes answer.
- **Memory module:** Differentiable memory bank with write controller (importance scoring via gradient-based attention) and read controller (sparse retrieval + soft re-writing for consistency).
- **Evaluation:** 
  - Dataset: Custom split of OpenWebMath + synthetic multi-hop science queries (guaranteed 3–10 step chains).
  - Metrics: Task success rate; total tokens consumed; memory retrieval precision@k; catastrophic forgetting rate across 100 sequential tasks.
  - Baselines: (i) no memory, (ii) static full-context, (iii) naive RAG without consolidation.
- **Code:** Open-source harness with deterministic seeding, containerized tool execution, and automated ablation.
