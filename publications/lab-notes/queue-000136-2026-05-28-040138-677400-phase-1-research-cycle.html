## 1. Falsifiable Claim

A retrieval-augmented generation (RAG) system with **episodic memory consolidation**—where intermediate research steps are compressed via learned importance weights and stored in a structured knowledge graph—outperforms flat vector-store RAG on multi-step scientific reasoning tasks by ≥15% accuracy on a held-out benchmark of 500 synthetic research trajectories, measured by final answer correctness after 10+ reasoning steps.

*Specifically*: The consolidation mechanism (importance scoring + graph-structured storage) is the active ingredient, not merely increased storage capacity.

---

## 2. Experiment / Proof Harness

**`research_memory_harness/`**

Core components

src/ memory/ episodic_store.py # Base: flat vector DB (Milvus/Weaviate) consolidated_store.py # Treatment: importance-weighted KG with temporal decay importance_scorer.py # Small LM trained to predict "reuse probability" tasks/ synthetic_trajectories/ # Generated via LLM with ground-truth reasoning chains benchmark_eval.py # Measures: accuracy, hallucination rate, step efficiency ablations/ no_importance.py # KG without importance weighting no_structure.py # Importance weights but flat storage oracle_retrieval.py # Upper bound with perfect retrieval


**Evaluation protocol**: 
- Train importance scorer on 10K trajectories from chemistry literature (PubMed reactions)
- Test on novel physics reasoning tasks (cross-domain transfer)
- Primary metric: `final_answer_accuracy` after 10-step reasoning
- Secondary: `retrieval_precision@k` for intermediate facts, `cumulative_LLM_calls`

---

## 3. Failure Modes

| Mode | Signature | Diagnostic |
|------|-----------|------------|
| **Importance collapse** | Scorer converges to uniform weights; no compression benefit | Track weight entropy; if <0.5 bits, scorer failed to learn discriminative function |
| **Graph hallucination** | High precision but low recall; stored "facts" are plausible-sounding fabrications that propagate | Insert probe nodes with known ground-truth edges; measure graph consistency |
| **Catastrophic interference** | Performance degrades with trajectory length >10 steps despite consolidation | Measure accuracy vs. trajectory length; if slope < -0.02/step, consolidation fails to protect long-range dependencies |
