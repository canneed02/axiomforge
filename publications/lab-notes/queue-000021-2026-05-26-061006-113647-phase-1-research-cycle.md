## 1. Falsifiable Claim

A retrieval-augmented generation (RAG) system with **episodic memory consolidation**—where intermediate research steps are compressed via gradient-based summarization and stored in a structured knowledge graph—outperforms append-only log baselines on multi-step scientific reasoning tasks by ≥15% on answer accuracy, measured after 100+ reasoning steps, when evaluated on a held-out suite of synthetic open problems (e.g., theorem proving, protocol design).

## 2. Experiment / Proof Harness

**SciMemBench**: A benchmark of 500 synthetic research tasks with verifiable ground truth.

| Component | Specification |
|-----------|-------------|
| **Task generator** | LLM-generated problems in formal domains (symbolic math, simple chemistry, algorithm design) with deterministic verifiers |
| **Memory architecture** | Baseline: append-only transcript; Treatment: transcript + compressed episodic memory (gradient-based key-value consolidation every *k* steps) + knowledge graph for entity relationships |
| **Evaluation metric** | Accuracy on final answer; memory retrieval precision/recall; cumulative token cost; steps to correct hypothesis |
| **Control** | Fixed reasoning budget (compute tokens); identical base model (e.g., Claude 3.5 Sonnet or Llama 3.1 70B) |

**Verification**: Open-source harness at `github.com/[org]/scimembench` with deterministic seeds, Dockerized execution, and automated regression testing.

## 3. Failure Modes

| **F3** | **Graph topology fragility**: Knowledge graph becomes disconnected or overloaded, causing reasoning chains to fail | Path length between related entities → ∞; query latency spikes; accuracy degrades superlinearly with task complexity |
