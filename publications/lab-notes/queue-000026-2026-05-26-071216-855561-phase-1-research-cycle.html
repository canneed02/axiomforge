## 1. Falsifiable Claim

A retrieval-augmented generation (RAG) system with **temporal decay-weighted retrieval** and **explicit contradiction logging** reduces cumulative factual drift in multi-step scientific reasoning by ≥30% compared to standard similarity-based retrieval, measured by contradiction rate in generated hypotheses after 100+ reasoning steps.

---

## 2. Experiment / Proof Harness

**System:** `research-memory-harness` (open-source benchmark)

| Component | Specification |
|-----------|---------------|
| **Corpus** | arXiv CS/ML papers (2019–2024), with injected synthetic contradictions |
| **Task** | Multi-hop literature synthesis: generate 200-step research threads |
| **Baseline** | Dense passage retrieval (DPR) with fixed top-k |
| **Treatment** | Temporal decay (λ=0.01/day) + contradiction-indexed sparse retrieval |
| **Metric** | Contradiction detection rate (human+LLM adjudicated) + downstream hypothesis accuracy on held-out evaluation questions |

**Validation:** Run 50 independent research threads; measure drift as internal inconsistency rate verified against source corpus.

---

## 3. Failure Modes

| ID | Failure Mode | Detection |
|:---|:---|:---|
| **F1** | **Decay overcorrection:** Recent but wrong sources dominate; temporal weighting suppresses foundational truths. | Compare against ablation with λ=0; measure recall of seminal papers. |
| **F2** | **Contradiction false positives:** System over-logs semantic near-duplicates as contradictions, causing retrieval collapse. | Audit logged contradictions; precision <80% triggers failure. |
| **F3** | **Adversarial memory injection:** Self-generated hypotheses re-ingested as "evidence" create feedback loops. | Track provenance chains; detect circular citation with graph analysis. |

## 4. Minimal Publishable Artifact

**Contradiction Audit Log (CAL) format** — a structured, versioned JSON-L schema for every retrieval event:

{ "query_id": "uuid", "retrieved": [{"source": "doi/arxiv_id", "timestamp": "ISO", "embedding": "hash"}], "contradictions_flagged": [{"pair": ["id_a", "id_b"], "confidence": 0.87, "adjudication": "pending"}], "temporal_weights": [0.91, 0.73, ...], "downstream_use": "hypothesis_id" }
