**Research Memo: Autonomous Research Memory**

## 1. Falsifiable Claim

A retrieval-augmented generation (RAG) system with **episodic memory consolidation**—where raw experiment logs are compressed into structured "insight summaries" via a learned compression policy—outperforms naive append-only RAG on downstream task accuracy after >1000 sequential experiments, measured by (a) retrieval precision for relevant prior results and (b) zero-shot transfer to modified experimental conditions.

*Specifically:* Consolidation reduces memory retrieval latency below O(n) in experiment count without dropping top-5 retrieval precision below 0.85 on held-out query sets.

---

## 2. Experiment / Proof Harness

**`memory_harness/`** — reproducible benchmark suite:

| Component | Specification |
|-----------|---------------|
| **Task generator** | Produces sequences of 1000+ synthetic experiments from SciFact + custom procedural domains; each experiment yields (hypothesis, protocol, result, unexpected_observation) tuples |
| **Memory variants** | (a) Naive vector DB append-only; (b) Rule-based periodic summarization; (c) **Target:** RL-trained compression policy (PPO on retrieval accuracy reward) |
| **Evaluation** | Held-out query set: 500 questions requiring cross-experiment synthesis; measure precision@5, latency, transfer accuracy on modified protocol variants |
| **Baseline** | Human-curated "gold" memory index for 100-experiment subset |

**Code artifact:** `github.com/[org]/memory_harness` — Dockerized, deterministic seeds, CI runs in <4 hours on single A100.

---

## 3. Three Failure Modes

| # | Failure Mode | Detection Signal |
|---|-------------|------------------|
| **F1** | **Catastrophic consolidation** — compressor over-generalizes, losing rare but critical failure modes | Precision drop >15% on "edge case" query subset; audit via saliency tracing to compressed memory slots |
| **F2** | **Retrieval illusion** — system retrieves plausible-sounding but hallucinated "past experiments" | Cross-reference with immutable raw log hashes; hallucination rate >2% triggers alarm |
| **F3** | **Staleness amplification** — consolidated memory encodes superseded assumptions, preventing update | Measure "concept drift detection": inject known protocol change; system fails to flag conflict within 10 experiments |

---

## 4. Minimal Publishable Artifact

**`consolidation_audit_trail`** — a single JSON-L file per experiment sequence containing:

{ "experiment_id": "exp_0847", "raw_log_hash": "sha256:abc...", "consolidation_input": ["exp_0847", "exp_0723", "exp_0691"], "consolidation_policy_version": "v0.3.1-4f2a9d", "produced_summary": "...", "retrieval_queries_where_used": ["q_1124", "q_1156"], "human_verified": false }


Enables: post-hoc analysis of *when* and *why* consolidation occurred, reproducibility without raw data exposure, and community auditing of failure modes F1–F3.
