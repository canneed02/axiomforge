**Research Memo: Autonomous Research Memory**

## 1. Falsifiable Claim

A retrieval-augmented generation (RAG) system with **episodic memory consolidation**—where intermediate research steps are compressed via learned importance weights and stored in a structured knowledge graph—outperforms flat vector-store RAG on multi-step scientific reasoning tasks by ≥15% accuracy on the GPQA and SciReason benchmarks, while using ≤50% of the storage tokens.

---

## 2. Experiment / Proof Harness

**"MemBench-Sci" Evaluation Protocol**

| Component | Specification |
|-----------|-------------|
| **Base model** | Llama-3.1-70B or equivalent |
| **Memory architecture** | Test two conditions: (a) flat RAG (Chroma/FAISS), (b) hierarchical RAG with learned consolidation |
| **Consolidation mechanism** | Trainable gating network scores step importance; low-importance steps are summarized, high-importance retained verbatim |
| **Knowledge graph** | Research steps as nodes; "depends_on", "refutes", "extends" edges |
| **Tasks** | GPQA diamond, SciReason multi-hop, synthetic 10-step reasoning chains |
| **Metrics** | Accuracy, retrieval precision@k, storage tokens, cascade error rate (step where first error occurs) |
| **Control** | Fixed compute budget; memory systems compared at equal retrieval latency |

**Code harness:** GitHub repo with Docker container, deterministic seeding, and `pytest` suite that fails if claim is violated.

---

## 3. Failure Modes

| ID | Failure Mode | Detection |
|:---|:---|:---|
| **F1** | **Consolidation collapses critical nuance** | Ablation: compare full-retention vs. consolidated on tasks requiring fine-grained distinction (e.g., protein variant effects); measure drop on "needle-in-haystack" subtasks |
| **F2** | **Graph topology degrades with scale** | Monitor path length distribution; if mean shortest path > threshold or connected components fragment, retrieval fails for distant dependencies |
| **F3** | **Importance weights overfit to training distribution** | Cross-domain evaluation: train on chemistry tasks, test on physics; if accuracy drop > flat RAG baseline, gating network lacks generalization |

---

## 4. Minimal Publishable Artifact

**"MemBench-Sci" benchmark dataset + evaluation kit**

- 5,000 synthetic multi-step research traces with ground-truth dependency graphs
- Annotated importance scores for each step (human + LLM-judge validated)
- Lightweight Python package: `pip install membench-sci`
- Includes pre-computed baselines and leaderboard submission format
- License: CC-BY-4.0; hosted on HuggingFace Datasets with versioned releases
