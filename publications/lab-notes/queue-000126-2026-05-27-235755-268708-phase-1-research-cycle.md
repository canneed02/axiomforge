def evaluate_citation_system(system_under_test, dataset):
    results = []
    for paper in dataset:
        query = generate_review_query(paper.abstract)
        generated_review = system_under_test.generate(query)
        
        citations = extract_cited_dois(generated_review)
        verified = verify_against_ground_truth(citations, paper.true_references)
        
        results.append({
            'precision': verified.correct / len(citations),
            'recall': verified.correct / len(paper.true_references),
            'hallucination_rate': verified.fabricated / len(citations),
            'confidence_calibration': brier_score(
                system_under_test.confidence_scores, 
                verified.ground_truth
            )
        })
    return aggregate(results)
```

## 3. Failure Modes

| ID | Failure Mode | Detection Signal |
|---|---|---|
| **F1** | **Retrieval collapse:** System over-relies on high-frequency but semantically generic chunks, missing domain-specific correct citations | Low citation diversity entropy; high overlap in retrieved chunks across dissimilar queries |
| **F2** | **Confidence miscalibration:** Model expresses high confidence for fabricated citations when retrieval gaps exist | Brier score > 0.25; systematic overconfidence in low-retrieval-score regions |
| **F3** | **Temporal hallucination:** System cites "future" papers or misattributes preprint versions due to version-agnostic retrieval | DOI-version mismatch; citation of papers with submission dates after query context cutoff |
