## 1. Falsifiable Claim

A retrieval-augmented generation (RAG) system with **episodic memory consolidation**—where intermediate research steps are compressed via learned importance weights and stored in a differentiable memory bank—outperforms static-context baselines on multi-step scientific reasoning tasks, measured by (a) reduced error accumulation rate and (b) lower token consumption, with statistical significance at *p* < 0.01 on held-out benchmark suites.

---

## 2. Experiment / Proof Harness

**"MemBench-Sci" Evaluation Protocol**

| Component | Specification |
|-----------|---------------|
| **Task** | Multi-hop scientific reasoning: given a seed paper, generate a valid experimental design, identify 3 confounders, propose statistical tests |
| **Baselines** | (i) Static 128k context, (ii) Naïve RAG with BM25, (iii) No memory (stateless) |
| **Treatment** | Episodic memory with: (a) importance scoring via gradient-based saliency, (b) memory consolidation via online clustering (k-means with learned representations), (c) retrieval with learned query-memory attention |
| **Metrics** | Error accumulation (deviation from ground-truth expert trace), tokens used, wall-clock time, memory bank hit rate for relevant prior steps |
| **Implementation** | Open-source harness at `github.com/[org]/membench-sci` with: fixed random seeds, containerized environment, automated regression against baselines |

---

## 3. Failure Modes

| # | Failure Mode | Detection Signal |
|---|--------------|----------------|
| **F1** | **Catastrophic memory interference**: Consolidation overwrites critical, infrequently-accessed memories (e.g., edge-case methodological constraints) | Sudden accuracy drop on long-tail reasoning subtasks; memory bank entropy collapse |
| **F2** | **Retrieval hallucination amplification**: Memory retrieval injects plausible but unverified "facts" from prior incorrect reasoning steps, creating self-reinforcing error chains | Trace back-propagated retrieval sources; measure contradiction rate between retrieved memory and ground-truth knowledge base |
| **F3** | **Importance scoring collapse**: Gradient saliency converges to uniform or path-dependent (not path-*independent*) importance, making memory equivalent to random sampling | Monitor importance distribution Gini coefficient; test invariance to presentation order of equivalent reasoning paths |
