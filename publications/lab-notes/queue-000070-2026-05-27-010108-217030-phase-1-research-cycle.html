## 1. Falsifiable Claim

A vector database with semantic chunking and LLM-generated metadata outperforms (lower retrieval precision@5 on scientific Q&A) a naive full-text embedding baseline for research artifact recall, *but only when query-to-document semantic drift exceeds a threshold measurable by embedding cosine similarity variance*.

---

## 2. Experiment / Proof Harness

**`membench/`**: Minimal reproducible benchmark

| Component | Specification |
|-----------|---------------|
| **Corpus** | 10k arXiv ML papers (2019–2024), chunked by section with citation graphs |
| **Baseline** | Naïve: embed full paper text, retrieve top-k by cosine similarity |
| **Treatment** | Structured: hierarchical chunks (claim/ evidence/ method/ result) with LLM-extracted entities, linked by citation edges |
| **Query set** | 500 manually-verified questions requiring cross-paper synthesis; annotated with drift score (variance of embedding similarities between query and relevant chunks) |
| **Metric** | Precision@5, stratified by drift quartile |

**Harness**: GitHub Actions workflow that spins up Weaviate/Milvus, ingests corpus, runs queries, outputs CSV + matplotlib figures. Single command: `make benchmark`.

---

## 3. Failure Modes

| # | Failure Mode | Detection |
|---|-------------|-----------|
| **F1** | **Semantic collapse**: High-drift queries map to same embedding subspace as low-drift; drift threshold becomes non-discriminating. Detect via UMAP visualization of query embeddings colored by actual vs. predicted drift quartile. |
| **F2** | **Metadata hallucination**: LLM-generated entities introduce false links; precision degrades on treatment despite better chunking. Detect by comparing human-annotated vs. LLM-extracted citation edges on 100-paper holdout. |
| **F3** | **Temporal staleness**: Research memory fails to invalidate superseded claims; retrieval surfaces deprecated methods. Detect by injecting 50 papers with explicit retraction/ supersession statements, measuring recall of invalidation edges. |

---

## 4. Minimal Publishable Artifact

- `harness/`: Python package with `pytest` tests for each failure mode
