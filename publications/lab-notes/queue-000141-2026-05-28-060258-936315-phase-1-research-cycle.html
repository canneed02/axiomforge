## 1. Falsifiable Claim

A retrieval-augmented generation (RAG) system with **contextual compression and explicit uncertainty quantification** outperforms naive vector-store retrieval for maintaining coherent multi-session scientific research trajectories, measured by (a) 30% lower topic drift in generated hypotheses across sessions and (b) 50% reduction in redundant experiment proposals, evaluated against a ground-truth trajectory graph annotated by domain experts.

---

## 2. Experiment / Proof Harness

**Harness: `research-memory-bench`**

- **Synthetic trajectories**: Generate 100 ground-truth research graphs (nodes: hypotheses/observations; edges: supports/refutes/modifies) using structured LLM prompts with manual validation on 10% sample.
- **Agent loop**: Run autonomous research agent for 20-step sessions; inject memory system under test.
- **Metrics**:
  - *Drift*: cosine distance between embedding of current focus and nearest ground-truth path node
  - *Redundancy*: Jaccard similarity of proposed experiments against prior proposals
  - *Recall*: fraction of ground-truth relevant prior nodes retrieved when needed
- **Baselines**: (i) naive top-k vector retrieval, (ii) full-context window (no retrieval), (iii) human-curated summary.
- **Code**: Python package with deterministic seeding, Dockerized, logging all retrievals and generations.

---

## 3. Failure Modes

| Mode | Description | Detection |
|------|-------------|-----------|
| **Retrieval collapse** | System fixates on high-frequency terms, missing rare but critical prior results | Track retrieval entropy; alarm if < 2 bits for >3 consecutive steps |
| **Confidence hallucination** | Uncertainty estimates poorly calibrated; system is "certainly wrong" | Measure calibration via Brier score on retrieval relevance predictions |
| **Graph erosion** | Long-term memory structure degrades as compressed summaries overwrite nuanced distinctions | Periodic audit: reconstruct ground-truth edge from memory, measure F1 |

- Open-source: 100 annotated synthetic trajectories, harness code, baseline implementations
