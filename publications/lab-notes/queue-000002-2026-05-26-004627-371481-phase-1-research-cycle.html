## 1. Falsifiable Claim

A retrieval-augmented generation (RAG) system with **episodic memory consolidation**—where intermediate research steps are compressed into summary vectors with explicit uncertainty scores—reduces error propagation in multi-step scientific reasoning by ≥30% compared to flat context windows, measured by downstream task accuracy on held-out benchmark suites (e.g., GPQA, SciREX).

---

## 2. Experiment/Proof Harness

**"MemTrace" Benchmark Protocol**

| Component | Specification |
|-----------|-------------|
| **Base model** | Fixed open-weight LLM (e.g., Llama-3-70B or Mistral-Large) |
| **Memory architecture** | Compare: (a) flat RAG, (b) hierarchical RAG, (c) episodic consolidation with uncertainty-weighted retrieval |
| **Task suite** | 500 multi-step scientific reasoning traces from chemistry/biology (synthesis planning, mechanism elucidation, literature synthesis) |
| **Metric** | End-to-end accuracy + **cascade error rate** (first error step index, if any) |
| **Control** | Fixed retrieval corpus; synthetic ground-truth reasoning graphs for each task |

**Implementation:** GitHub repo with deterministic seeding, containerized evaluation, and automated regression against baseline.

---

## 3. Failure Modes

| ID | Failure Mode | Detection Signal |
|----|-------------|----------------|
| **F1** | **Consolidation-induced hallucination**: Memory compression invents non-existent experimental results or citations | Cross-reference retrieved chunks against source corpus; measure citation verifiability rate |
| **F2** | **Uncertainty collapse**: Uncertainty scores become miscalibrated (overconfident) after recursive self-updates | Track expected calibration error (ECE) of uncertainty scores across memory generations |
| **F3** | **Retrieval fixation**: System repeatedly retrieves stale memories, missing paradigm shifts in recent literature | Inject synthetic "paradigm shift" papers; measure time-to-adoption in reasoning traces |
