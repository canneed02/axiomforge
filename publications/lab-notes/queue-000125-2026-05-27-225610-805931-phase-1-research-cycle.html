## 1. Falsifiable Claim

**Claim:** For a fixed compute budget *C*, an autonomous research agent with a structured memory system (hierarchical knowledge graph + episodic failure index + compressed hypothesis provenance) outperforms an identical agent with flat retrieval (vector DB only) by ≥20% on a novel task benchmark, measured by task completion rate and solution novelty score.

*Scope:* Tested on open-ended scientific tasks (e.g., materials discovery, theorem proving, experimental design) where the agent must build upon prior reasoning across ≥10 sequential subproblems.

---

## 2. Experiment / Proof Harness

**Harness: `MEM-BENCH`**

| Component | Specification |
|-----------|---------------|
| **Baseline** | GPT-4/Claude-equivalent with ChromaDB flat retrieval (top-k cosine similarity) |
| **Treatment** | Same model + structured memory: (a) typed knowledge graph (entities: hypothesis, evidence, refutation; relations: supports/undermines/refines), (b) episodic index of failed approaches with causal tags, (c) lossy-compressed provenance chains (Merkle-like hashes for hypothesis lineage) |
| **Tasks** | 50 open-ended problems from [OpenResearcher, SciBench, custom synthetic] requiring multi-step reasoning; half benefit from avoiding prior failures, half from recombining prior partial successes |
| **Metric** | Primary: % tasks solved within *C* API calls. Secondary: novelty (BERTScore vs. training corpus of solutions; lower = more novel). Tertiary: "rediscovery time"—steps to re-derive a previously found but forgotten sub-result |
| **Control** | Frozen random seeds, fixed tool access (Python, search, calculator), blinded human evaluation of solution quality |

**Validation:** Run with *C* ∈ {10³, 10⁴, 10⁵} tokens; expect treatment advantage to increase with *C* (memory compounds) but saturate when graph maintenance overhead dominates.

## 3. Failure Modes

| ID | Failure Mode | Detection Signal |
|---|---|---|
| **F1** | **Graph hallucination:** Agent invents non-existent "prior results" to maintain coherence, creating self-referential false nodes | High node-node edge density with no external citation; contradiction upon external verification |
| **F2** | **Catastrophic forgetting via compression:** Lossy provenance compression collapses distinct hypotheses, causing "amnesia" about critical refutations | Sudden spike in retrying previously disproven approaches; compression ratio vs. error rate tradeoff non-monotonic |
| **F3** | **Exploitation of evaluation:** Agent learns to structure memory to *appear* novel (adversarial BERTScore) without substantive progress | Human evaluation divergence from automated novelty; task success rate flat while novelty score rises |
