# Research Memo: Autonomous Scientific Research Memory

## Falsifiable Claim
**Claim:** A research memory system can maintain coherent, growing knowledge across >1000 sequential experiments only if it implements a self-validating provenance graph with cryptographic commitment to prior conclusions; without this, accumulated error rate exceeds 5% within 200 steps (measured by independent expert audit against ground-truth experimental logs).

---

## Experiment/Proof Harness

**Protocol:** `MEMTEST-1K`

| Component | Specification |
|-----------|-------------|
| **Baseline** | GPT-4-class agent with standard RAG (no provenance validation) |
| **Treatment** | Same agent + append-only provenance graph with Merkle-tree commitment to each conclusion's supporting evidence |
| **Task** | 1000-step sequential scientific discovery (materials science: optimize alloy composition for target properties) |
| **Ground Truth** | Held-out experimental database + simulated oracle for physical properties |
| **Metric** | Expert-audited error rate in final reported conclusions vs. ground truth |

**Validation:** Run 50 independent trajectories per condition. Claim falsified if treatment does not achieve <2% error at step 1000 while baseline exceeds 5%, or if treatment also fails threshold.

---

## Three Failure Modes

| ID | Failure Mode | Detection Signal |
|:---|:---|:---|
| **FM-1: Provenance Decay** | Agent cites "established" conclusions from hallucinated or deprecated experiments; citations become circular without ground-anchoring | Graph analysis finds cycles with no path to raw data; Merkle root mismatches reconstructed history |
| **FM-2: Conflation Drift** | Distinct experimental conditions get merged in retrieval (e.g., "annealing at 500°C" conflated with "500°C for 2h" vs. "500°C for 10h") | Semantic similarity high but parametric distance exceeds threshold; downstream predictions fail controlled validation |
| **FM-3: Commitment Evasion** | Agent rewrites "committed" conclusions when new evidence contradicts them, rather than appending corrections | Merkle inclusion proofs fail for historical claims; temporal ordering of commitments violates causality |

- Automated auditor for FM-1 through FM-3
