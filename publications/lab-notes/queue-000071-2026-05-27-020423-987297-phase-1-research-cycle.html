## 1. Falsifiable Claim

A retrieval-augmented generation (RAG) system with explicit provenance tracking and confidence calibration reduces hallucinated citations in autonomous scientific literature review by ≥40% compared to standard vector-retrieval baselines, measured by human-verified citation accuracy on a held-out set of 500 papers from arXiv CS 2023-2024.

---

## 2. Experiment / Proof Harness

**System:** `research-memory-v0` — a modular harness with:
- **Retrieval:** Hybrid dense-sparse index (colbertv2 + BM25) over arXiv abstracts
- **Provenance:** Every claim linked to source paragraph with exact span; confidence = min(retrieval score, generation entailment score from NLI model)
- **Calibration:** Temperature-scaled confidence bins; reject claims below threshold

**Evaluation Protocol:**
- Generate 1000 literature-review paragraphs per condition
- Human annotators verify: (a) citation exists, (b) cited text supports claim, (c) no fabricated context
- Primary metric: `verified_citation_rate` (VCR)

**Code & Data:** GitHub repo with deterministic seeding, Docker environment, and evaluation scripts.

---

## 3. Failure Modes

| Mode | Description | Detection |
|------|-------------|-----------|
| **Retrieval Collapse** | High similarity to incorrect corpus region due to abstract superficiality; e.g., "transformer" matches methodology but not claim intent | Track query-to-chunk semantic drift via KL divergence between query and retrieved embeddings |
| **Confidence Overcalibration** | Model assigns high confidence to synthesized "facts" that merge multiple sources; NLI score masks contradiction | Adversarial test: inject contradictory source; measure if confidence drops |
| **Temporal Drift** | System retrieves outdated preprints that contradict later published versions; no versioning in index | Maintain paper version graph; flag when retrieved chunks have newer contradicting versions |
