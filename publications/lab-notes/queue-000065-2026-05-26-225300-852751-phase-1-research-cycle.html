## 3. Failure Modes

| Mode | Symptom | Diagnostic |

| **Catastrophic consolidation** | Correct intermediate conclusions overwritten by wrong compressed summaries | Trace accuracy vs. compression ratio; should show cliff, not smooth decay |
| **Uncertainty miscalibration** | High-uncertainty retrievals still used; model overconfident in wrong summaries | Expected calibration error (ECE) on held-out verification tasks >> 0.1 |
| **Retrieval collapse** | All queries return same few summaries; semantic diversity metric collapses | Measure effective retrieval set size; should remain >k/2 for k-retrieval |
