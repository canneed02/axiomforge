## 1. Falsifiable Claim

**Claim:** For a fixed compute budget, an autonomous research agent with a structured memory system (hierarchical knowledge graph + episodic replay + working memory buffer) outperforms an identical agent with only raw context window or flat vector retrieval by ≥15% on a novel scientific benchmark requiring multi-step reasoning across non-contiguous research episodes, measured by task completion rate and solution novelty score.

---

## 2. Experiment / Proof Harness

**Harness:** `ARES-Bench` (Autonomous Research Evaluation Suite)

| Component | Specification |
|-----------|-------------|
| **Baseline** | GPT-4-class model with 128K context, no memory structure |
| **Control** | Same model + flat RAG (top-k chunks) |
| **Treatment** | Same model + hierarchical memory (knowledge graph + episodic replay + working memory) |
| **Task** | 50 synthetic open problems in an unseen domain (e.g., "Design a protocol for X" where X∉ training data), requiring accumulation of intermediate findings across 10-100 sub-episodes |
| **Metric** | Primary: % tasks solved to criterion; Secondary: novelty score (cosine distance to nearest training solution) |
| **Budget** | Fixed API cost per task ($5-$50 scaling) |

**Code artifact:** GitHub repo with deterministic task generator, memory implementations, evaluation pipeline, and reproducible run scripts.

---

## 3. Failure Modes

| # | Failure Mode | Detection |
|---|-------------|-----------|
| **F1** | **Memory bloat:** Knowledge graph accumulates contradictions; retrieval precision decays below random baseline after N episodes. Detect via precision@k on synthetic ground-truth queries. | Track retrieval precision vs. episode count |
| **F2** | **Epistemic overfitting:** Agent replays stale hypotheses, fails to update on disconfirming evidence. Detect via "belief revision score" on injected contradictory findings. | Inject contradictions; measure update rate |
| **F3** | **Competence illusion:** High novelty score from superficial recombination, but solutions fail physical/logical validation. Detect via automated theorem prover or simulator check. | Validate outputs with domain-specific verifier |
