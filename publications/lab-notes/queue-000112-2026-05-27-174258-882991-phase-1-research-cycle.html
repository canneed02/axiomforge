## 1. Falsifiable Claim

A retrieval-augmented generation (RAG) system with episodic memory—indexed by (hypothesis, outcome, confidence) triples—outperforms identical systems using only document corpora on novel scientific reasoning tasks, measured by 10% higher accuracy on held-out inference chains requiring cross-paper synthesis.

---

## 2. Experiment / Proof Harness

**Memory-Enabled Scientific Reasoning Benchmark (MESRB)**

- **Corpus**: 1,000 synthetic papers with controlled citation graphs and embedded methodological flaws
- **Task**: Given a research question, generate a valid experimental design; score requires citing ≥2 papers with non-obvious conflict resolution
- **Memory condition**: System accumulates (hypothesis, outcome, confidence) from 50 prior "experiments" on related sub-problems
- **Control**: Same system, same corpus, memory wiped between queries
- **Metric**: Valid design rate (human-verified), citation precision/recall, conflict detection F1

Implementation: GitHub repo with deterministic paper generation, Dockerized evaluation, and automated judge (GPT-4 with structured rubric, calibrated against human experts on 100 examples).

---

## 3. Failure Modes

| Mode | Mechanism | Detection |
|------|-----------|-----------|
| **Confidence collapse** | Episodic entries converge to high confidence on early hypotheses, suppressing exploration | Track entropy of retrieved memory distributions over task sequence |
| **Synthetic citation** | Memory fabricates plausible-sounding but non-corpus "findings" | Ground every retrieval to corpus UUID; flag unverifiable memories |
| **Task drift accumulation** | Memory from sub-problem A poisons reasoning on distant sub-problem B | Measure cosine drift between query embedding and retrieved memory context; abort if >threshold |

---
