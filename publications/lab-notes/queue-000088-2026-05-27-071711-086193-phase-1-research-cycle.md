## 1. Falsifiable Claim

A retrieval-augmented generation (RAG) system with explicit provenance tracking and confidence calibration exhibits measurably lower hallucination rates on scientific literature synthesis tasks compared to identical architectures without such memory scaffolding, with the effect size increasing with task duration (>24h autonomous operation).

## 2. Experiment / Proof Harness

**HARNESS-MEM-001**

| Component | Specification |
|-----------|-------------|
| **Baseline** | Standard RAG (LangChain/LlamaIndex default) with vector DB |
| **Treatment** | Identical base + (a) source provenance chains, (b) confidence scoring per retrieval, (c) explicit contradiction logging, (d) periodic consistency checks |
| **Task** | 72-hour autonomous literature review on evolving topic (e.g., "CRISPR off-target effects 2023-2024") |
| **Evaluation** | Human expert audit of 100 randomly sampled claims against source documents; measure precision, recall, hallucination rate, contradiction detection rate |
| **Control** | Same system with memory modules disabled; same query stream |

**Metrics:** Hallucination rate (primary), source attribution accuracy, self-contradiction rate, drift from initial hypotheses.

---

## 3. Failure Modes

| ID | Failure Mode | Detection Signal |
|----|-------------|------------------|
| **FM-1: Confidence Collapse** | Confidence scores drift to extremes (over/under-confidence) after iterative self-query; system becomes uncalibrated | Track calibration curve (Brier score) over time; flag if expected calibration error >0.1 |
| **FM-2: Source Entanglement** | Retrieved chunks from distinct papers merge into synthetic "sources" that don't exist; provenance chains become fiction | Periodic exact-match verification of cited DOIs/paragraphs against original corpus |
| **FM-3: Archive Amnesia** | Early findings overwritten by recency-biased retrievals; system loses track of abandoned hypotheses that were correct | Audit trail of hypothesis lifecycle; measure "regret" vs. oracle with full history |

---

## 4. Minimal Publishable Artifact

**`memory_audit_log.jsonl`** — Append-only, cryptographically signed (SHA-256) record of every retrieval, confidence score, source snippet, and reasoning step. Schema enforces: timestamp, query, retrieved chunks (with hashes), generated claim, confidence, contradiction flags, and parent claim IDs. Enables full reproducibility and external audit without exposing proprietary model weights.
