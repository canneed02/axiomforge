## 1. Falsifiable Claim

A retrieval-augmented generation (RAG) system with **episodic memory consolidation**—where raw experiment logs are compressed into structured "insight summaries" via a learned compression policy—outperforms naive append-only RAG on downstream task accuracy after >1000 sequential experiments, measured by (a) retrieval precision for relevant prior results and (b) zero-shot transfer to modified experimental conditions.

*Specifically:* Consolidation reduces memory retrieval latency below O(n) in experiment count without dropping top-5 retrieval precision below 0.85 on held-out query sets.

## 3. Three Failure Modes

| # | Failure Mode | Detection Signal |
|---|-------------|------------------|
| **F1** | **Catastrophic consolidation** — compressor over-generalizes, losing rare but critical failure modes | Precision drop >15% on "edge case" query subset; audit via saliency tracing to compressed memory slots |
| **F2** | **Retrieval illusion** — system retrieves plausible-sounding but hallucinated "past experiments" | Cross-reference with immutable raw log hashes; hallucination rate >2% triggers alarm |
| **F3** | **Staleness amplification** — consolidated memory encodes superseded assumptions, preventing update | Measure "concept drift detection": inject known protocol change; system fails to flag conflict within 10 experiments |
